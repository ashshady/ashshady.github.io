1151 LCA in a Binary Tree

五月 3, 2021 发布在 pat, 甲级

PAT甲级题集，最低公共祖先问题

The lowest common ancestor (LCA) of two nodes U and V in a tree is the deepest node that has both U and V as descendants.

Given any two nodes in a binary tree, you are supposed to find their LCA.

Input Specification:

Each input file contains one test case. For each case, the first line gives two positive integers: M (≤ 1,000), the number of pairs of nodes to be tested; and N (≤ 10,000), the number of keys in the binary tree, respectively. In each of the following two lines, N distinct integers are given as the inorder and preorder traversal sequences of the binary tree, respectively. It is guaranteed that the binary tree can be uniquely determined by the input sequences. Then M lines follow, each contains a pair of integer keys U and V. All the keys are in the range of int.

Output Specification:

For each given pair of U and V, print in a line LCA of U and V is A. if the LCA is found and A is the key. But if A is one of U and V, print X is an ancestor of Y. where X is A and Y is the other node. If U or V is not found in the binary tree, print in a line ERROR: U is not found. or ERROR: V is not found. or ERROR: U and V are not found..

Sample Input:

6 8
7 2 3 4 6 5 1 8
5 3 7 2 6 4 8 1
2 6
8 1
7 9
12 -3
0 8
99 99

Sample Output:

LCA of 2 and 6 is 3.
8 is an ancestor of 1.
ERROR: 9 is not found.
ERROR: 12 and -3 are not found.
ERROR: 0 is not found.
ERROR: 99 and 99 are not found.

思路分析

既然是二叉树，就一定要运用好中序有序这个条件，为所有的节点按照中序访问的遍历标上序号，这个在读入中序序列的时候就可以做到，然后按照前序中序建树的代码改出来：若当前根节点标号大于两节点，说明两节点在当前根节点的左子树，下一层递归应是左子树；反之若当前根节点标号小于两根节点，说明两节点在当前根节点的右子树，下一层递归应是右子树；若当前根节点标号在两节点的中间，说明已经找到答案；若当前根节点等于其中一个节点的标号，说明其中一个节点是另一个节点的祖先

AC代码

#include <bits/stdc++.h>

using namespace std;

const int inf = 0x3f3f3f3f;
const int minInf = -0x3f3f3f3f;
const int maxn = 210;
const int mod = (int) 1e9 + 7;
typedef long long ll;
typedef unsigned long long ull;

vector<int> pre, in;
int l, r;
int l_pos, r_pos;
map<int, int> pos;

void LCA(int preL, int inL, int inR) {
    if (inL > inR)
        return;
    int root = pre[preL];
    int root_pos = pos[root];
    if (root_pos > l_pos && root_pos > r_pos)
        LCA(preL + 1, inL, root_pos - 1);
    else if (root_pos < l_pos && root_pos < r_pos)
        LCA(preL + root_pos - inL + 1, root_pos + 1, inR);
    else if ((root_pos > l_pos && root_pos < r_pos) || (root_pos > r_pos && root_pos < l_pos))
        printf("LCA of %d and %d is %d.\n", l, r, root);
    else if (root_pos == l_pos)
        printf("%d is an ancestor of %d.\n", l, r);
    else if (root_pos == r_pos)
        printf("%d is an ancestor of %d.\n", r, l);
}

int main() {
    int n, query;
    cin >> query >> n;
    int t;
    for (int i = 0; i < n; ++i) {
        cin >> t;
        in.push_back(t);
        pos[t] = i;
    }
    for (int i = 0; i < n; ++i) {
        cin >> t;
        pre.push_back(t);
    }
    while (query--) {
        cin >> l >> r;
        if (pos.count(l) && pos.count(r)) {
            l_pos = pos[l];
            r_pos = pos[r];
            LCA(0, 0, n - 1);
        } else {
            if (!pos.count(l) && !pos.count(r))
                printf("ERROR: %d and %d are not found.\n", l, r);
            else if (!pos.count(l))
                printf("ERROR: %d is not found.\n", l);
            else
                printf("ERROR: %d is not found.\n", r);
        }
    }


    return 0;
}
